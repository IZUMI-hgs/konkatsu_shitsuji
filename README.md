# サービス詳細
 
そんなときに役立つのが、
婚活の'お相手'管理サービス、
『婚活執事』です。

『婚活執事』は'お相手'に関わる以下のような情報を保存しておき、
婚活をスムーズに進めることができます。
・名前
・趣味
・仕事
・休日
・年収
・アレルギーの有無
・デートの回数
・前回デートの内容
・前回デート時の私の服装、髪型
・前回のデートの話題
・好きポイント
・う〜んポイント
お相手と会ったその日に婚活執事からの質問に答えて行くだけで
お相手の情報が記録されます。
例え次のデートが1ヶ月後でも、きちんと情報が管理されているので安心です。

また、素敵な"執事"が頑張るあなたを応援してくれたり
ワンポイントアドバイスをしてくれる事もポイントです。
